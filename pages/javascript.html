    Variables

    Los lenguajes de programación reciben revisiones. Estas revisiones se le llama ECMA script

    ¿Que variables usamos?

    Usaremos las variables a partir del ECMAscript6


    Covenciones

    1.UpperCamelCase: Significa que cuando nosotros creemos una variable, la primera letra que vayamos a crear de esa variable tiene que ser con mayúscula y en el caso de que exista otra palabra dentro de esa variable, esa debera contener otra mayúscula.

    2.camelCase: Por el contrario nos exige que cada palabra debe comenzar con mayúscula pero sin la primera palabra.

    3.snake_case: En cambio esta convención nos exige que cada palabra debera estar separada por un guión bajo y en minúsculas.

    Tipos de datos

    JavaScript tiene siete tipos de datos primitivos, siendo String uno de ellos. En JavaScript, una cadena representa una secuencia de caracteres y puede estar entre comillas simples (') o dobles (").
    Tenga en cuenta que las cadenas son inmutables, lo que significa que una vez creadas, no se pueden cambiar.
