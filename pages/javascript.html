Tipos de datos

JavaScript tiene siete tipos de datos primitivos, siendo String uno de ellos. En JavaScript, una cadena representa una secuencia de caracteres y puede estar entre comillas simples (') o dobles (").
Tenga en cuenta que las cadenas son inmutables, lo que significa que una vez creadas, no se pueden cambiar.
